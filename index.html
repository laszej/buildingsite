Witam na mojej stronie internetowej!

HTML, CSS oraz JavaScript. Niektóre z nich są są dosyć proste, inne wymagały sporo cierpliwości i czasu jak np. "Amazorron", który będąc dość złożoną aplikacją, napisałem bardzo imperatywnym kodem JS, ale też mocno testując przy okazji swoje umiejętności manipulowania Document Object Model poprzez JavaScript. W przyszłości do tworzenia tego typu projektów będę korzystał z React.js, który jest moim aktualnym kierunkiem rozwoju, i który znacznie przyśpiesza i ułatwia budowanie złożonych aplikacji.
