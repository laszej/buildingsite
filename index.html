Witam na mojej stronie internetowej!

  Witryna składa się z dwóch części. Pierwszą jest portfolio, w którym umieszczam stworzone przez siebie projekty. Na dzień dzisiejszy, wszystkie są napisane w "czystej triadzie" tj. HTML, CSS oraz JavaScript. Kolejność tych projektów jest (prawie) przypadkowa. Niektóre są banalne, inne wymagały sporo cierpliwości i czasu jak np. "Amazorron", który będąc dość złożoną aplikacją, napisałem bardzo imperatywnym kodem JS, ale też mocno testując przy okazji swoje umiejętności manipulowania Document Object Model poprzez JavaScript. Bardziej złożone projekty będę tworzył oczywiście korzystając z frameworków(bibliotek) gdyż doskonale rozumiem, jak ważna jest nie tylko jakość, ale również możliwie jak najszybsze stworzenie aplikacji. Przechodząc na bardziej "korporacyjne" technologie jak np. React, będę je w miarę możliwości - jakie zapewnia github pages - umieszczał również tutaj. Jeśli "statyczne" gh pages nie wystarczy, będę linkował do projektów na innych hostingach.
